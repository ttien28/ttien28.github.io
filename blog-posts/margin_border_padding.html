Technical Blog: Margin, Border and Padding

August 17th, 2014

Margin: The margin is the amount of transparent space between one element and the next. Type "margin: x" where x is the amount of desired space to be added, expressed as a length, a percentage of the width of the parent element, or auto.

Border: A border can be created around an element or on individual sides of an element and set its thickness, style, and color. Type "border-style: type" where type is none, dotted, dashed, solid, double, groove, ridge, inset, or outset.

Padding: This is extra space around the contents of an element but inside the outer edge of the border. It can be expressed in any combination of pixels, percentages, ems or rems.
